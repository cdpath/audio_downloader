# Dictionary Audio Downloader

Chrome extension to download audio from dictionary websites.

## Installation

Clone and build:

```bash
git clone git@github.com:cdpath/audio_downloader.git
cd audio_downloader
npm install
npm run build
```

Load in Chrome:

1. Open `chrome://extensions/`
2. Enable Developer mode
3. Click Load unpacked
4. Select `.output/chrome-mv3` folder generated in previous step
